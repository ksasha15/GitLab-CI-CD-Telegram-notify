# Git && ChatOps
- сделать интеграцию с GitLab CI/CD.
- Настроить оповещение об успешном/не успешном завершении сборки pipeline в чат Telegram.

#### В задании запускается bash скрипт с условием сравнения входящей переменной (if/else). В зависимости от выполнения условий отправляется соответствующее сообщение в чат Telegram.
